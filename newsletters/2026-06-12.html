🧠 Code Reviews now learn your repo’s standards
REVIEWS.md gives your review agent repo-specific instructions in plain Markdown. Add your conventions once, then Kilo applies them during automated GitHub and GitLab reviews.
Code Review Memory closes the loop. Kilo can analyze how your team responds to review feedback, propose updates to REVIEWS.md, and even open the PR for you.
Read about REVIEWS.md → Open Code Reviews →
🧑‍💻 Agent Manager runs agents in parallel
Every agent gets its own git worktree, branch, and repo copy. Run multiple tasks at once without agents stepping on each other’s files.
It all lives inside Kilo Code. Start runs from the multi-agent sidebar, track them on a board, review the diff, and merge when ready.
Read about Agent Manager →
🤖 KiloClaw keeps getting easier
Recent upgrades make setup faster and daily use smoother. Morning briefs, @kiloclaw.ai forwarding, one-click ClawBytes, simpler Calendar OAuth, and Composio support are all here.
The goal is useful on day one. Your Claw can summarize the day, catch urgent messages, prep around calendar conflicts, and remember what matters.
See what’s new in KiloClaw →
💳 Coding Plans are live
You can now buy external token plans with your Kilo balance. Plans work across Kilo Code products and Kilo Gateway, starting with MiniMax M3.
It’s built for bigger agent work. MiniMax M3 brings a 1M context window, multimodal input, and generous monthly usage for concurrent runs.
Read about Coding Plans → View plans →
🖥️ Kilo Console beta is live
Kilo Console is a browser UI for the local Kilo CLI. Manage projects, worktrees, sessions, agents, tools, permissions, providers, notifications, and themes from one place.
The beta is ready for feedback. Install the release candidate, open a project, launch sessions, and tell us what feels great or rough.
