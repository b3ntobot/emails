📱 New launch: Kilo is now on mobile

Kilo already runs in VS Code, the CLI, and the cloud. Now it runs on your phone. Android is live today on Google Play, and iOS is in App Store review with a waitlist open.

☁️ Cloud Agents just got a major upgrade
Kilo's cloud coding agents are now more personal, configurable, and connected. Connect your GitHub account so every commit and PR the agent opens is authored by you with the Kilo bot as co-author, bundle env profiles with your custom agents, skills, MCP servers, and slash commands, and watch PR status without leaving the Kilo dashboard.
Remote sessions tie it all together. Turn on remote mode in the CLI and your active local sessions show up in the web interface next to your cloud sessions, with messages and responses syncing in real time. Start something locally, check in from your phone, hand it off to a cloud session when you close the laptop. The compute stays on your machine, the access is everywhere.

🏢 What's new for enterprise: six things from Focus Week
For Enterprise, AI adoption is so last year — focus is on visibility and control. During Focus Week, one group concentrated on enterprise requests and shipped six updates aimed at giving security and platform teams real answers about how agents are being used.

💸 Powerful AI coding without frontier-model prices
You can run Kilo well on $0/month — and even a paid setup doesn't have to hurt. A new guide from Brendan walks through three real budgets: $0, $5–15, and $20+, with the exact models and providers to use at each tier.
The headline move is Auto Efficient. It classifies each request in real time and routes to the cheapest model that KiloBench has proven accurate enough for that specific task. Routine work runs lean, hard work gets more firepower, and the router keeps adjusting as the session evolves. Auto Free is the zero-config path for $0.

One more way to route requests through the Kilo gateway. The Anaconda Desktop provider is now available in the model picker, joining the long list of first-party and BYOK options already wired into Kilo Code.
